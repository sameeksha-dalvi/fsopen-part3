# Full Stack Open - Part 3

This repository contains my solutions for **Part 3 (Programming a server with NodeJS and Express)** from the Full Stack Open course by the University of Helsinki.

## What I Learned

In this part I learned how to build a backend server using Node.js and Express and connect it to MongoDB.

Key concepts covered:

- Building REST APIs with Express
- Middleware in Express
- Connecting a backend to MongoDB using Mongoose
- Environment variables using dotenv
- Backend validation with Mongoose schemas
- Error handling in Express
- Using ESLint to enforce code quality

## Phonebook Backend

The main project in this part is a **Phonebook API** that allows users to:

- Add a new contact
- View all contacts
- Update contact information
- Delete contacts

The backend stores the data in **MongoDB**.

## Technologies Used

- Node.js
- Express
- MongoDB
- Mongoose
- ESLint
